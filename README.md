# 🏥 Hospital Management System

A modern and responsive Hospital Management System developed using Django framework.  
This project helps hospitals and clinics manage patients, doctors, appointments, and administrative operations efficiently through a centralized web application.

---

## 🚀 Features

### 👨‍⚕️ Doctor Module
- Doctor registration & login
- Manage patient records
- View appointments
- Update consultation details

### 🧑‍🤝‍🧑 Patient Module
- Patient registration & authentication
- Book appointments online
- View doctor availability
- Access treatment details

### 🛠️ Admin Module
- Secure admin dashboard
- Manage doctors & patients
- Approve appointments
- Monitor hospital activities

### 📅 Appointment System
- Online appointment booking
- Appointment status management
- Schedule tracking

### 🎨 User Interface
- Responsive design
- Clean dashboard UI
- Bootstrap integrated frontend
